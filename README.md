# UDP client-server application

## Pre-requisites 
- CMake 
- G++ compiler
- Boost
- OpenCV

## User Instructions 
Link Boost and OpenCV root and libraries manually if they are downloaded in a non-standard path
```
$ git clone https://github.com/changdaozheng/udp_messaging.git
$ cd udp_messaging
$ mkdir build
$ cmake --build build
$ build/server_client
```
